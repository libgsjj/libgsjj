Detailed Description

template<typename Sigma>
class gsjj::DFA< Sigma >

A Deterministic Finite Automaton.

A DFA is encoded by a 5-tuple with:

  • , the set of states. Here, a state is represented by an integer (starting from 0)
  • , the alphabet. Here, the alphabet is all the values that can be coded in a given C++ type (for example, a char)
  • , the initial state
  • , the transition function such that delta:
  • , the set of final/accepting states

We make the assumption that cannot be empty (because there must be at least the initial state).

Adds a transition to the DFA.

If the DFA is in the state p and reads the given symbol, it goes to the state q. The function checks that delta(p, symbol) does not yet exist and that p and q exist.

Parameters
pThe state where the transition starts
symbolThe symbol to read to activate the transition
